Pearson Distributions
=========================

pearson **dist**\ ributions----------------------------------------------------------
``pearsondist``
*Release* v1.0

The ``pearsondist`` Python package is designed to construct Pearson distributions by matching
moments of the unknown true distributions. Currently, we are able to match the first eight
moments.
